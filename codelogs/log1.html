Code Log 1: CSS Measuring Units

Absolute units are fixed and not affected by other elements or the user's settings. They are best used when you want elements to have a specific, unchanging size. An example of an absolute unit would be px, which is pixels that are 1/96 of 1 inch.

Relative units are based on other values, such as the size of the parent element, the root element, or the viewport. They are flexible and scale better across different devices and screen sizes.
