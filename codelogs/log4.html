Code Log 4 - CSS Color

CSS uses color to differentiate html elements, and it allows us to control the appearance of content by applying colors to different content such as text, background, and borders. Color values in CSS allow you to change the color of elements such as text an example of this is: p { color: red; } You can also change the color of the background an example of this is: div { background-color: lightblue; } You can also change the colore of a border: div { border: 2px solid; border-color; green: }
